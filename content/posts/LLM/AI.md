MCP 是 ModelContextProtocol

RAG 检索增强生成（Retrieval Augmented Generation）是是一种机器学习技术，它通过检索相关数据来增强生成。

RPC 远程过程调用（Remote Procedure Call）是一个分布式计算技术，它允许应用程序通过网络进行通信。
jsonrpc 是一种用于在网络中传输数据的协议，它使用 JSON 格式来传输数据。

RESTful API 是目前比较成熟的一套互联网应用程序的 API 设计理论
